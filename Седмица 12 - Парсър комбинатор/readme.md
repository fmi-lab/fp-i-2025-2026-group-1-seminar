# Седмица 12 - Парсър комбинатор

## Задача
Напишете алгебричен тип данни `Parser`, представляващ комбинатор за парсване на изрази, състоящ се от една единствена функция, която приема символен низ и връща стойност от тип `Either`, която или съдържа грешка при парсването на подадения символен низ, или съдържа наредена двойка от парснатата стойност и остатъка от низа, която не е била обработена все още.

За `Parser` да се направят инстанции на класовете `Functor`, `Applicative`, `Monad` и `Alternative`.

Да се напишат следните помощни функции:

- `satisfy :: (Char -> Bool) -> Parser Char` - връща Parser, който обработва един знак, отговарящ на подадения предикат
- `char :: Char -> Parser Char` - връща Parser, който обработва произволен знак
- `string :: String -> Parser String` - връща Parser, който обработва цял символен низ
- `digit :: Parser Char` - връща Parser, който обработва една цифра
- `integer :: Parser String` - връща Parser, който обработва едно цяло число
- `whitespace :: Parser String` - връща Parser, който обработва множество последователни `whitespace` знаци
- `separateBy :: Parser a -> Parser b -> Parser [b]` - приема Parser, обработващ някакъв разделител и Parser обработващ един елемент и връща Parser обработващ последователност от елементи, разделени с разделителя

С помощта на тези функции да се напише функция, която обработва JSON обекти на студенти, подобни на тези, разгледани в Седмица 06.
